All-Star Game Selection History: San Diego Padres

My hometown baseball team had 5 players selected for the 2024 Major League Baseball All-Star Game. I was curious about how the amount of selections compared to history and how many selections were in the All-Star Game’s starting lineup. Plotly’s Hover Text provided an opportunity to view player names without having to make a table.

Most of this project involved accessing the Lahman Database through the Lahman package. I had to make a small dataframe to include data for 2023 and 2024 seasons, as the Lahman Database only included All-Star Game data up to 2022. If were to to make a similar project for all 30 Major League teams, I would probably look at scraping Baseball-Reference.com for 2023 and 2024 data (to avoid making manually making a dataframe for each team).



Data sources:

Lahman, S. (2023) Lahman’s Baseball Database, 1871-2022, 2022 version, https://www.seanlahman.com/baseball-archive/statistics

MLB.com (n.d.) San Diego Padres All-Stars, https://www.mlb.com/padres/history/all-stars

MLB.com (2024) 2024 All-Star Game Rosters, https://www.mlb.com/news/2024-all-star-game-rosters

Code:

library(Lahman)
library(tidyr)
library(dplyr)
library(stringr)
library(ggplot2)
library(plotly)
library(ggthemes)
PlayersSince2022 <- data.frame(yearID = c(2023,2023,2024,2024,2024,2024,2024),
                           startingPos = c(NA,NA, NA, NA, 9, NA, NA),
                           fullName = c("Josh Hader", "Juan Soto", "Luis Arraez", "Jackson Merrill", 
                                        "Jurickson Profar", "Robert Suarez", "Fernando Tatis"))
all_star_games_team<- AllstarFull  |> 
  filter(teamID == "SDN" )|> 
  select(yearID, playerID, teamID, startingPos) |> 
  left_join(People, by = "playerID") |> 
  select(yearID, playerID, nameFirst, nameLast,teamID, startingPos) |> 
  mutate(fullName = paste0(nameFirst, " ", nameLast)) |> 
  bind_rows(PlayersSince2022) 
all_star_games_team$starter <- ifelse(all_star_games_team$startingPos %in% 1:10, "yes", "no")
df_summary <- all_star_games_team |> 
  group_by(yearID) %>%
  summarize(
    Starter = sum(starter == "yes"),
    Non_Starter = sum(starter == "no")
  ) %>%
  pivot_longer(cols = c(Starter, Non_Starter),
               names_to = "type",
               values_to = "count")
all_star_games_team <- all_star_games_team |> 
  group_by(yearID) |> 
  summarize(
    Starter = paste(fullName[starter=="yes"], collapse = ", "), 
    Non_Starter = paste(fullName[starter=="no"], collapse = ", ")
  ) 

all_star_games_team <- all_star_games_team |> 
  pivot_longer(cols=c(Starter, Non_Starter),
               names_to="type",
               values_to = "names")

merged_df <- merge(df_summary, all_star_games_team, by = c("yearID", "type")) |> 
  mutate(type = ifelse(type == "Non_Starter", "Non-Starter", type)) 


p <- ggplot(merged_df, aes(factor(yearID), y = count, fill=type, 
                           text = paste0("Year: ", yearID, "<br>",
                                         ifelse(type == "Starter", "Starter: ", "Non-Starter: "),
                                         names)))+
  geom_bar(stat = "identity") +
  scale_fill_manual(values = c("Starter" = "#FFC425", "Non-Starter" = "#2F241D"),
                    labels = c("Starters", "Non-Starters")) +
  labs(x = "Year", y = "Number of Players", fill = "Player Type",
       title = "San Diego Padres All-Star Game Selection History", caption = "Data sources: Lahman, S. (2023) Lahman’s Baseball Database, 1871-2022, 2022 version, https://www.seanlahman.
com/baseball-archive/statistics/ \n https://www.mlb.com/padres/history/all-stars")+
  scale_y_continuous(breaks = seq(0, 10, by = 1))+
  theme_minimal()+
  theme(plot.title = element_text(hjust = 0.5))+
  theme(panel.grid.major = element_blank(), panel.grid.minor = element_blank())+
  theme(axis.text.x = element_text(angle = 90, hjust = 1, vjust = 0.5))
plotlyViz <- ggplotly(p, tooltip = "text") |> 
  layout(title = list(text = paste0('All-Star Game Selection History',
                                    '<br>',
                                    '<b>San Diego Padres<b>',
                                    '<br>'))) 
config(plotlyViz, displayModeBar = FALSE, showLink = FALSE)
